# Тести: перформанс/UX/SEO

## Мета
Перевірити, що сторінка реагує плавно, секції відкриваються з анімацією, та заповнені базові SEO/OG/Schema мета-дані.

## Передумови
- `public/index.html` відкрито (з мок-даними або Firebase).

## Кроки
1) Intersection Observer / reveal
   - Прокрутити сторінку. Очікування: секції з класом `reveal` плавно з'являються (opacity/translate). Без стрибків розмітки.
2) Модалки
   - Відкрити/закрити кілька модалок поспіль. Очікування: ESC/✕/клік поза працюють стабільно.
3) Lazy-load
   - Переконатися, що великі зображення мають `loading="lazy"` (інспектор). При повільному з'єднанні не блокує рендер.
4) Навігація
   - Клік по навігаційним якорям. Очікування: плавний скрол до секції без різких стрибків.
5) SEO/OG/Schema
   - Інспектором перевірити наявність meta `og:title/description/image/url/type`, `twitter:card`, `theme-color`, JSON-LD `MusicArtist` зі `name`/`genre`/`url`/`sameAs`.

## Результат
- Pass, якщо всі пункти виконані без консольних помилок/варнінгів.
