# Тест: додавання відео без YouTube ID

## Мета
Переконатися, що відео можна додати без YouTube ID, картка з'являється на фронті, а модалка показує повідомлення замість плеєра.

## Кроки
1) Увійти як адмін (UID у ADMIN_UIDS, firebaseConfig заповнений).
2) У формі Videos заповнити лише Title, залишити поле "YouTube ID (опційно)" порожнім, за бажанням додати прев'ю.
3) Натиснути "Додати".
4) Очікування:
   - Картка відео з'являється одразу в гріді, без перезавантаження.
   - Під назвою картки видно підказку "YouTube ID не вказано".
   - Клік по картці відкриває модалку з текстом "YouTube ID не вказано" (без плеєра).
5) Видалити тестову картку — вона зникає одразу.

## Результат
- Pass, якщо картка додається та відкривається без плеєра, показуючи попереджувальний текст, і видаляється одразу.
