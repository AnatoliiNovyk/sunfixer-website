# Тести: адмін UI/UX (overlay, тости, прев'ю)

## Мета
Перевірити, що адмін-панель виглядає та працює як повноцінний дашборд: окремий оверлей, кнопка закриття, тости успіху/помилок, прев'ю завантажень.

## Кроки
1) Вхід
   - Натиснути "⚙️ Admin" → залогінитись як UID з ADMIN_UIDS.
   - Очікування: тост "Адмін-режим активний", бейдж статусу онлайн, бейдж користувача з email/uid.
2) Відкриття/закриття панелі
   - Кнопка "⚙️ Admin" відкриває оверлей; кнопка "Закрити" ховає його.
   - Очікування: панель не перекриває фронт, але має окремий фон/scroll.
3) Прев'ю завантажень
   - У будь-якій формі dropzone вибрати зображення.
   - Очікування: під дропзоною з'являється прев'ю/назва файлу.
4) CRUD + тости
   - Додати реліз/відео/фото/подію/партнера з валідними полями.
   - Очікування: тост успіху; при відсутності обов'язкових полів — alert + беззмінний стан.
5) Видалення
   - Видалити елемент будь-якої колекції.
   - Очікування: елемент зникає, помилку видно через alert/toast якщо rule не пускає.
6) Логаут
   - Натиснути "Вийти".
   - Очікування: панель закрита, статус "Увійдіть".

## Результат
- Pass, якщо overlay/кнопки/тости/прев'ю працюють як описано й CRUD/видалення дають зворотний зв'язок.
