# Тести: модалка `#modalOverlay`

## Мета
Перевірити, що єдиний оверлей відкриває контент для релізів, відео та фото і закривається усіма шляхами.

## Передумови
- Запущено `public/index.html` у браузері (локально або через простий `http-server`).

## Кроки
1) Клік по демо-картці релізу в секції Discography.
   - Очікування: відкривається модалка з заголовком "Placeholder Release" і Spotify iframe.
2) Клік по демо-картці відео в секції Videos.
   - Очікування: модалка з YouTube iframe, попередній контент замінено.
3) Клік по демо-фото (будь-якому) в секції Photos.
   - Очікування: модалка з зображенням Unsplash та заголовком.
4) Закриття модалки:
   - Клік по ✕ → закриває.
   - Клік по напівпрозорому фону → закриває.
   - Натиснути ESC → закриває.
5) Після закриття жодна модалка не відображається, сторінка скролиться як звичайно.

## Результат
- Pass, якщо всі очікування виконані; інакше зафіксувати крок і симптом.
   - Особливо перевірити клік по напівпрозорому фону: має закривати модалку.
