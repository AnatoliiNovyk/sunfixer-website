# Тест: нормалізація gs:// та CORS fallback

## Мета
Переконатися, що gs:// URL зі старим доменом firebasestorage.app автоматично нормалізуються до appspot.com і зображення відображаються без CORS/404.

## Кроки
1) У Firestore залишити/створити запис із `cover_gs_url` або `image_gs_url`, що містить `gs://sunfixer-website.firebasestorage.app/...` (старий формат).
2) Відкрити лендінг через локальний сервер (http://localhost:8080/).
3) Очікування: зображення завантажується (немає CORS/404), карточка відображає прев'ю.
4) У Network немає червоних рядків для цього ресурсу; URL, який реально запитується, містить `appspot.com` або `storage.googleapis.com`.

## Результат
- Pass, якщо картка показує зображення і немає CORS/404 у консолі.
