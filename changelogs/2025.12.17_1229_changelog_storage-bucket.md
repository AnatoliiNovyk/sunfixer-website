# Changelog 2025.12.17_1229 — storage bucket fix

## Зміни
- Замінено `storageBucket` у `public/index.html` на стандартний формат `sunfixer-website.appspot.com`, щоб уникнути CORS/404 для файлів зі Storage.
- Оновлено версію Service Worker кешу до `sunfixer-cache-v5` для примусового підвантаження актуальних ассетів після зміни конфігу.

## Як було
- Використовувався bucket `sunfixer-website.firebasestorage.app`, запити на зображення отримували 404/CORS.
- SW міг тримати закешовану стару версію `index.html`.

## Як стало
- Storage звертається до `sunfixer-website.appspot.com`; очікується коректний доступ до файлів без CORS.
- Новий SW-кеш завантажить свіжий `index.html` та ассети.

## Причина
- Усунути CORS/404 при завантаженні зображень із Firebase Storage.
