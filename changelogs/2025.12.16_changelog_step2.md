# Changelog 2025.12.16 — Каркас лендінгу

## Було
- Репозиторій без реалізації інтерфейсу та без структури `public/`.

- Додано `public/index.html` з каркасом лендінгу: 8 секцій, базова кіберпанк тема (Orbitron/Rajdhani, неонова палітра), навігація, кнопка Admin, єдина модалка `#modalOverlay` з керуванням ESC/✕/клік поза.

- Функціональні тести не додавались на цьому кроці (каркас UI без логіки даних). Будуть створені окремі тестові файли разом з функціоналом на наступних кроках.
