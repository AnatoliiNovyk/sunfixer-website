# Changelog — скрипти локального http-server

## Що зроблено
- Додано скрипти для швидкого запуску локального сервера з каталогу `public/`:
  - `scripts/start-server.ps1` (PowerShell) — пробує `python -m http.server` або `npx serve`.
  - `scripts/start-server.sh` (bash) — аналогічно використовує python3/python або `npx serve`.
- Оновлено README з посиланням на скрипти для зручного старту.

## Як було
- Було лише текстове пояснення про ручний запуск `python -m http.server`/`npx serve` без готових скриптів.

## Як стало
- Є готові скрипти для Windows та Unix/macOS; README містить інструкцію запуску.

## Причина змін
- Спрощення ручного тестування PWA/Service Worker та уникнення помилок CORS при відкритті напряму з файлової системи.
