# Changelog — інструкція локального http-server

## Що зроблено
- Додано секцію в README про запуск локального http-server для коректного тестування PWA/Service Worker та уникнення CORS.
- Описано два варіанти без додаткових залежностей: `python -m http.server 8080` (від кореня, шлях `/public/`) або `npx serve public -l 8080`.

## Як було
- README радив відкривати `public/index.html` напряму, без пояснення про обмеження SW/PWA при файловому відкритті.

## Як стало
- Є чітка інструкція, як підняти локальний сервер і з нього відкрити сторінку, щоб працювали SW/PWA і не було CORS.

## Причина змін
- Забезпечити відтворювані умови для ручних тестів PWA/Service Worker та уникнути проблем при запуску з файлової системи.
