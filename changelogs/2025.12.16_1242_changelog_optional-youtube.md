# Changelog 2025.12.16_1242 — відео без YouTube ID

## Зміни
- Зроблено поле YouTube ID опційним у формі відео; додано підказку замість плеєра, якщо ID відсутній.
- Модалка відео тепер показує текст "YouTube ID не вказано" коли немає ID, без помилкових iframe.
- Грід відео відображає попередження під назвою для записів без ID.
- Додано тестовий сценарій перевірки додавання/видалення відео без ID.

## Як було
- Поле YouTube ID було обов'язковим; відсутність ID блокувала додавання відео.
- Модалка завжди намагалася вставити iframe, що вимагало валідного ID.

## Як стало
- Відео можна додати без YouTube ID; картка з'являється одразу, модалка показує повідомлення.
- Користувач бачить, що ID не вказано, і може додати його пізніше.

## Причина
- Дозволити публікувати відеоконтент, коли посилання на YouTube ще недоступне.
