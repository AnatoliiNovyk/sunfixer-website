# Changelog 2025.12.16 — Модалка та демо-контент

## Було
- Каркас сторінки без робочих модалок для карток релізів/відео/фото.

## Стало
- Додано демо-тригери для модалки у секціях Discography, Videos, Photos.
- Єдиний `#modalOverlay` тепер відкриває Spotify/YouTube/фото-контент і закривається через ESC/✕/клік поза.
- Підготовлено демо-дані (release/video/photo) для вбудованих iframe/зображень.
- Додано тестовий файл [tests/modal-behavior.md](../tests/modal-behavior.md) з інструкціями перевірки модалки.

## Причина
- Виконання Кроку 3 плану: реалізувати поведінку модалок та продемонструвати сценарії для релізів/відео/фото.

## Примітки/тести
- Ручні тести описані у `tests/modal-behavior.md`; автоматизація не додавалася на цьому етапі.
