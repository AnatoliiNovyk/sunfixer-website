# Changelog 2025.12.16 — PWA доповнення, іконки PNG, rules

## Було
- PWA вже була, але без PNG-іконок 192/512, кеш SW мінімальний, правила Firestore лише у README без блоку в коді.

## Стало
- Згенеровано PNG-іконки [public/icon-192.png](../public/icon-192.png) та [public/icon-512.png](../public/icon-512.png); додано в [public/manifest.webmanifest](../public/manifest.webmanifest) поряд із SVG.
- Оновлено [public/sw.js](../public/sw.js): нова версія кешу `v2`, кешування manifest/icon, stale-while-revalidate для зображень/ CDN (fonts/Google), cache-first для shell.
- Додано довідковий блок Firestore Rules біля `firebaseConfig` у [public/index.html](../public/index.html); README має приклад правил.

## Причина
- Виконати додаткові покращення: повний набір PWA іконок, розширений кеш SW, явна документація правил Firestore.

## Примітки/тести
- Перевірка PWA/іконок/офлайн — за чеклістом [tests/pwa.md](../tests/pwa.md).
- Після зміни SW версії потрібен перезапуск/refresh з оновленим SW.
